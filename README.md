# Generative AI Research

This repository contains code and data artifacts from our generative AI research project. Data files
are collected in [data/](/data), while analysis scripts (since there are fewer of them) live in the
repository root.

The final report is being written as a [Typst](https://typst.app/) document, and is in
[report/](/report), along with the bibtex references.

## Contributing

Further data files should be added to [data/](/data), and analysis scripts to the project root. If
you're a member of the research team, supplemental documents should be uploaded to the shared folder
on Onedrive.

Since this is a public repository, no sensitive personal data should be committed to this
repository. Upload this to Onedrive as well.

_Note for readers: This is not actually a generative AI research project, but a collection of loose
resources to demonstrate folder layout for a research methods course._
